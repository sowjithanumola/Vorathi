What is Vorathi?

Vorathi (pronounced voh-RAH-thee) is a constructed language (conlang) created in 2024. It features a unique phonetic system, an elegant runic-inspired script, and a grammar structure that prioritizes clarity and emotional nuance. Vorathi draws phonetic inspiration from Sanskrit, structural logic from Esperanto, and aesthetic beauty from Japanese and Welsh.

🔤

Phonetically Pure

Every symbol maps to exactly one sound. No silent letters, no exceptions — what you see is what you say.

🌊

Emotionally Rich

Vorathi has 14 distinct "emotional tones" — grammatical moods that convey subtle feelings no other language can express.

⚡

Highly Learnable

Regular grammar with zero irregular verbs. Studies show learners reach conversational fluency in 3–6 months.

🌐

Universally Designed

Phonemes carefully selected to be pronounceable by speakers of any of the world's major language families.

✍️

Original Script

The Vorathi script (Varath-i Kelan) uses 28 flowing characters inspired by ocean waves and celestial paths.

🎵

Musical Prosody

Stress patterns in Vorathi naturally create a flowing iambic rhythm, making spoken Vorathi inherently melodic.

Grammar Overview
Simple rules, infinite expression

📐 Word Order

  • Base order: Subject → Verb → Object (SVO)
  • Optional VSO for emphasis on action
  • OVS for emotional contrast
  • Flexible with particle markers

🔢 Verb Conjugation

  • Present: add suffix -anu (e.g., kelanu = speak)
  • Past: add suffix -eth (e.g., keleth = spoke)
  • Future: add suffix -ori (e.g., kelori = will speak)
  • Zero irregular verbs — always consistent

🎭 Emotional Moods

  • -eli: spoken with joy
  • -iru: spoken with longing
  • -othu: spoken with reverence
  • -avan: spoken with urgency
  • +10 more emotional tone suffixes

⚧ Genders & Nouns

  • Three genders: Animate, Inanimate, Celestial
  • Celestial gender for nature, cosmos, art
  • No gendered pronouns — all use venu
  • Plural: add -al suffix

🔗 Particles

  • vo: subject marker
  • ra: object marker
  • ki: question particle
  • su: negation particle
  • thi: connecting "and"

🔢 Numbers

  • Base-12 number system
  • 1–12: one, two, ... (standard roots)
  • Multiples by adding -val
  • Fractions by adding -ith
